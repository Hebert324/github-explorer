## Sobre o projeto:

Github explorer e um website para encontrar informações sobre repositórios do GitHub, a onde podemos ver o número de issues, Forks, Commits feitos é muito mais....

## :rocket: Tecnologias utilizadas:

- React JS
- API do GitHub
- Styled Components
