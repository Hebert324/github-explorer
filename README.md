## 📌 Sobre o Projeto

O **GitHub Explorer** é um site que permite **buscar repositórios públicos** do GitHub e visualizar detalhes como:
- ⭐ Estrelas
- 🍴 Forks
- 🐛 Issues abertas
- 📦 Commits
- 📄 Informações completas de cada repositório

Ideal para desenvolvedores que desejam acompanhar ou explorar projetos interessantes diretamente da interface web.
